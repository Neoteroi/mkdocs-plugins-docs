# Plugins for MkDocs

Plugins for [MkDocs](https://www.mkdocs.org) and [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/)
by [Roberto Prevato](https://twitter.com/RobertoPrevato).

```shell
pip install neoteroi-mkdocs
```
